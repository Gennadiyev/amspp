Class MultiModalDataset Multi-modal dataset. A multi-modal dataset takes multiple torch.utils.data.Dataset (or any iterable) objects and create a formatted output for downstream tasks.
Class Schema A Schema is a data structure that transforms from raw data or tensors to a specific format. It works similarly to the collate_fn argument in PyTorch's DataLoader class. The main difference is that a schema is not a function, but a class that can be instantiated and called.
Function schema A decorator that converts a function into a Schema object.
